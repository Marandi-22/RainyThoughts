# Hero's Journey: Ultimate Abusive Enemy System
## Complete Integration of Personal Tormentors + AI-Powered Psychological Warfare

## Project Overview
Create the most brutally honest text-based RPG where enemies represent your worst critics, bullies, and inner demons. They use AI-generated dialogue based on your actual goals, problems, and weaknesses with zero boundaries - pure psychological warfare designed to fuel your revenge motivation through productivity.

## Core Mechanics

### Two-Layer System
1. **Pomodoros = Story/Quest Progression + EXP**
   - Each 25-minute session advances the narrative
   - Fights enemies, unlocks psychological areas, gains base EXP
   - Story progresses regardless of what you're doing during the session

2. **Task Completion = Stat Point Allocation**
   - Check off completed tasks from your todo list
   - Choose which stat to boost when marking tasks complete
   - Can complete tasks outside of Pomodoro sessions (like gym work)
   - Each completed task = stat points to allocate

### Four Power Categories with Enemies' Strengths/Weaknesses
- 💰 **WEALTH** (Money/business tasks)
- 💪 **STRENGTH** (Work/gym/physical tasks) 
- 🧠 **WISDOM** (College/study/learning tasks)
- 🎯 **LUCK** (Miscellaneous/random tasks)

## AI Dialogue Generation System

### Smart Caching with Personal Data Integration
```javascript
const shouldRegenerateDialogue = (enemy, userData) => {
  const currentDataHash = generateDataHash(userData);
  
  return (
    !enemy.dialogues.when_weak.length || // No dialogues yet
    enemy.lastDataHash !== currentDataHash || // Data changed
    userData.newJournalEntries > 0 || // New problems added
    userData.goalsUpdated || // Goals modified
    userData.statsChanged || // Stat improvements/declines
    enemy.dialogues.when_weak.length < 8 // Need more variety
  );
}

const generatePersonalizedDialogues = async (enemy, userData) => {
  if (!shouldRegenerateDialogue(enemy, userData)) {
    return enemy.dialogues; // Use cached
  }
  
  const dialogues = await generateAbusiveDialogues(enemy, userData);
  enemy.dialogues = dialogues;
  enemy.lastDataHash = generateDataHash(userData);
  
  // Save to persistent storage
  saveEnemyData(enemy);
  return dialogues;
}
```

## Enhanced Battle System

### Pre-Battle with Real-Time AI Generation
```javascript
const startBattle = async (enemyKey, userData) => {
  const enemy = enemies[enemyKey];
  
  // Generate or retrieve personalized taunts
  const dialogues = await generatePersonalizedDialogues(enemy, userData);
  
  // Select appropriate taunt based on current state
  const selectedTaunt = selectContextualTaunt(enemy, dialogues, userData);
  
  // Display battle screen with personalized psychological warfare
  displayBattleScreen(enemy, selectedTaunt, userData);
}

const selectContextualTaunt = (enemy, dialogues, userData) => {
  const totalStats = userData.wealth + userData.strength + userData.wisdom + userData.luck;
  const isStrong = totalStats > 300 && userData.streakDays > 7;
  const isWeak = totalStats < 100 || userData.streakDays < 3;
  
  if (enemy.alwaysArrogant && isStrong) {
    return randomFrom(dialogues.always_arrogant);
  } else if (isStrong) {
    return randomFrom(dialogues.when_afraid);
  } else {
    return randomFrom(dialogues.when_weak);
  }
}
```
